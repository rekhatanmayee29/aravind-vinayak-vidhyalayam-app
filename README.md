# Aravind Vinayak Vidhyalayam App

This project is a simple school management application designed to help the teacher and chairman of Aravind Vinayak Vidhyalayam track student progress.

## Features
- Student records management
- Attendance tracking
- Activity uploads
- Progress reports for chairman

## Users
Teacher – enters student data, attendance, and activities.

Chairman – monitors progress and reports.

## Future Goals
- Mobile friendly interface
- Offline functionality
- Smart dashboard analytics
